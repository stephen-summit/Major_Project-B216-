🚀 HarmoniqMind – AI-Powered Mental Wellness Assistant

🧠 Overview

HarmoniqMind is an AI-powered mental wellness platform designed to help users manage stress, 
emotions, and decision-making. It provides intelligent conversational support, mood-based 
insights, and personalized guidance during challenging situations.

🎯 Features
🤖 AI-powered chatbot for emotional support
📊 Real-time mood analysis
💡 Smart suggestions for stress & anger management
🔒 Secure and privacy-focused system
🎯 Decision-making assistance during emotional situations
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / React
Backend: Node.js, Express
Database: MongoDB
AI/NLP: Intelligent response system

📦 Installation & Setup
# Clone the repository
git clone https://github.com/stephen-summit/Major_Project-B216-.git

# Navigate to project
cd harmoniqmind

# Install dependencies
npm install

# Run the project
npm start
🔐 Environment Variables

Create a .env file and add:

MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
